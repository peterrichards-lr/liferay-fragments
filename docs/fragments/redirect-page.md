# Redirect Page

The **Redirect Page** fragment provides a declarative way to navigate users to a new URL after a specific event, such as a form submission.

## Key Features

- **URL Configuration**: Support for both internal Liferay pages and external URLs.
- **Delay Control**: Option to delay the redirection for a specified number of seconds.
- **Form Submission Hook**: Automatically triggers after a Liferay form is successfully submitted.

## Configuration

- **Target URL**: The destination page.
- **Delay (ms)**: Time to wait before redirecting.
