# Search Bar

A clean, modern search input integrated with Liferay Search, optimized for
placement in site headers.

## Visuals

![Search Bar](../../docs/images/header-search-bar.png)

## Key Features

- **Meridian Ready**: Built using theme tokens for consistent spacing and color.
- **Accessible**: Follows Liferay accessibility standards.
- **Responsive**: Mobile-first design.
