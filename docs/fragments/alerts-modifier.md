# Alerts Modifier

The **Alerts Modifier** fragment is a specialized component within the Liferay
Fragments library.

## Overview

A utility to enhance Liferay Alerts and Announcements with custom styling.

## Key Features

- **Meridian Ready**: Built using theme tokens for consistent spacing and color.
- **Accessible**: Follows Liferay accessibility standards.
- **Responsive**: Mobile-first design.
