# Dynamic Collection Slider

A high-performance slider that utilizes Liferay Collections, dynamically rendering items with multiple layout options.

## Features
- **5 Layout Styles**: Top Image, Full Background, Editorial Split, Inset Overlay Card, and Spotlight Avatar.
- **Advanced Navigation**: Touch swipe, mouse drag, and keyboard arrow key support.
- **Visual Depth**: Optional "Coverflow-lite" 3D effect for active slides.
- **Intelligent Image Extraction**: Automatically finds images from featured image slots, custom fields, or even embedded `<img>` tags in rich text.
- **Responsive**: Dynamic adjustment of visible slides based on device width.

## Configuration
- **Collection ID**: The Name, Key, or UUID of the Liferay Collection.
- **Slides per View**: Number of slides visible on desktop.
- **Display Style**: Choose between the 5 distinct layouts.
- **Enable Depth Effect**: Toggle the 3D active slide highlighting.
- **Slide Height**: Manually adjust the vertical height of slides.
- **Autoplay**: Configurable timing and looping options.
