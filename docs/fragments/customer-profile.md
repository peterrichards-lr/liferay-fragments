# Customer Profile

The **Customer Profile** fragment is a specialized version of the profile view, optimized for external-facing portals and B2B/B2C commerce scenarios.

## Key Features

- **Account Context**: Displays information relevant to the user's Commerce Account.
- **Service Integration**: Links to order history, support tickets, and case management.
- **Polished UI**: Features high-fidelity cards and action buttons.

## Configuration

- **Primary Action URL**: Destination for the main CTA (e.g., "Edit Profile").
- **Show Account ID**: Toggle visibility of the Commerce Account identifier.
