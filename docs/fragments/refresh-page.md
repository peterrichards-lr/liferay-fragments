# Refresh Page

The **Refresh Page** fragment is a utility that reloads the current page, useful for clearing form states or updating dynamic content.

## Key Features

- **Automatic Trigger**: Can be configured to refresh the page after a successful form submission.
- **Configurable Delay**: Prevents immediate refresh to allow user feedback (e.g., success messages) to be seen.

## Configuration

- **Delay (ms)**: Time to wait before refreshing the page.
- **Trigger Event**: The event that initiates the refresh.
