# Video Hotspots

## Overview

The Video Hotspots fragment allows you to overlay interactive, animated hotspot links on top of a background video element. This is useful for shoppable videos, interactive tutorials, or guided tours.

## Configuration

- **Video URL**: The source URL for the background video (MP4 or WebM).
- **Hotspots**: Configure up to 3 hotspots. For each hotspot, you can define:
  - Enable/Disable toggle
  - X and Y Position (percentage relative to video size)
  - Tooltip Text
  - Destination Link

## Usage

Add this fragment to any page where you want to feature interactive video content. Use the configuration panel to position the hotspots visually.
