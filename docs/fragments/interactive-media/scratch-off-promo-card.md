# Scratch-Off Promo Card

## Overview

The Scratch-Off Promo Card fragment provides an engaging, interactive marketing tool. It features a top canvas layer that users can "scratch off" using their mouse or touch device to reveal promotional content hidden underneath.

## Configuration

- **Overlay Color**: A solid color hex code to use as the scratchable layer (used if no image is provided).
- **Overlay Image URL**: An optional image to cover the promo content.
- **Brush Size**: The size (radius in pixels) of the scratching tool.
- **Clear Threshold**: The percentage of the area that must be scratched before the entire layer automatically fades away.

## Usage

Drag and drop any Liferay fragment or content into the drop zone provided by this fragment. The content will be obscured by the canvas layer until the user interacts with it.
