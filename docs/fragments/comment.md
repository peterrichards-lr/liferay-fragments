# Object Comment

The **Object Comment** fragment adds a simplified commenting interface to any Liferay Object record view.

## Key Features

- **Headless Integration**: Submits comments directly to the Liferay Object's comment endpoint.
- **Real-time Feedback**: Updates the UI immediately after a comment is successfully posted.
- **Themed Design**: Styled to match modern social interaction components.

## Configuration

- **Max Length**: Enforces a character limit on comments.
- **Placeholder**: Hint text for the comment input.
