# Segmented Numeric Input

The **Segmented Numeric** fragment is a high-fidelity UI component for capturing codes, PINs, or fixed-length numeric sequences.

## Key Features

- **Auto-Focus Flow**: Automatically moves the cursor to the next segment as the user types.
- **Restricted Input**: Enforces numeric-only entries.
- **Accessibility**: Grouped fieldset with individual ARIA labels for each digit.

## Configuration

- **Segment Count**: The number of digits/boxes to display.
- **Label**: Descriptive text for the grouped input.
