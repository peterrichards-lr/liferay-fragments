# Card Content

The **Card Content** fragment is a Smart Component designed to be nested inside a **Primary Card** or **Secondary Card**.

## Key Features

- **Adaptive Styling**: Automatically adjusts its internal typography, colors, and link styles based on whether it is placed inside a Primary or Secondary card.
- **Clean Structure**: Provides pre-styled slots for Title, Description, and an Action Link.

## Usage

1.  **Nest**: Drop this fragment into the dropzone of a Primary or Secondary Card.
2.  **Configure**: Fill in the text and link details. The styling will update automatically based on the parent container.
