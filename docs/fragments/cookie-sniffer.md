# Cookie Sniffer

The **Cookie Sniffer** fragment is a utility component that detects the presence of specific browser cookies and triggers actions based on their values.

## Key Features

- **Conditional Logic**: Execute JavaScript or show/hide other fragments based on cookie data.
- **Privacy Conscious**: Only accesses cookies explicitly configured by the Page Editor.
- **Liferay Integration**: Can be used to personalize content based on session or preference cookies.

## Configuration

- **Cookie Name**: The name of the cookie to monitor.
- **Match Value**: The value to look for within the cookie.
