# Pulse Button

The **Pulse Button** is a high-fidelity Call-to-Action (CTA) component that is
pre-integrated with Liferay Analytics Cloud and Pulse tracking logic.

## Key Features

- **Campaign Tracking**: Automatically includes campaign IDs and UTM parameters
  in the event payload.
- **Themed UI**: Modern button design with hover animations and Meridian token
  support.
- **Conversion Tracking**: Optimized for measuring CTA performance and user
  journey milestones.

## Configuration

- **Button Text**: The label shown on the button.
- **Action Type**: Choose between navigation (link) or event trigger.
- **Goal Category**: Categorize the conversion for reporting in Analytics Cloud.
