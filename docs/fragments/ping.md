# Ping (Connectivity Test)

The **Ping** fragment is a diagnostic utility that verifies connectivity between the client browser and the Liferay Headless APIs.

## Key Features

- **Latency Tracking**: Measures the round-trip time for API requests.
- **Success/Error States**: Visual indicators for connection status.
- **Silent Mode**: Can be used as a background keep-alive or health check.

## Configuration

- **Interval (ms)**: How often to perform the connectivity test.
- **Endpoint URL**: The specific API path to ping.
