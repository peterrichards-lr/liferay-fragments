# Submit Button (Custom)

The **Submit Button** fragment provides an enhanced submission experience with loading states and success/error animations.

## Key Features

- **Interactive States**: Transitions to a loading spinner or checkmark upon submission.
- **Form Integration**: Specifically designed to work with Liferay Page Fragments and Forms.
- **Themed Branding**: Full control over background and hover colors via configuration or Meridian tokens.

## Configuration

- **Label**: The text displayed on the button.
- **Loading Text**: Text shown while the submission is in progress.
- **Submit Color**: Primary button color.
