# Date Display (Collection Display)

The **Date Display (Collection Display)** fragment is specifically designed for
use within Liferay Collection Providers to format and display dates from mapped
content or object fields.

## Key Features

- **Field Mapping**: Direct integration with Liferay's collection mapping
  interface.
- **Localized Formatting**: Uses standard browser Intl.DateTimeFormat for
  language-appropriate date strings.
- **Relative Dates**: Option to show dates as "2 days ago" or "Next week".

## Configuration

- **Date Field**: The mapped field containing the date value.
- **Format**: Choose from short, medium, long, or full date styles.
