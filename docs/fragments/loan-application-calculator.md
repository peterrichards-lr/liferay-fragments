# Loan Application Calculator

The **Loan Application Calculator** fragment is an advanced financial tool that
combines a repayment calculator with a pre-qualification workflow.

## Key Features

- **Amortization Calculation**: Real-time calculation of monthly payments based
  on loan amount, interest rate, and term.
- **Workflow Integration**: Can be linked to a Liferay Object or Workflow to
  capture "Apply Now" events.
- **Interactive Sliders**: Smooth user interaction for exploring different loan
  scenarios.

## Configuration

- **Max Loan Amount**: Upper limit for the loan slider.
- **Default Interest Rate**: The base annual percentage rate (APR).
- **Target Object**: The Object where application data should be stored upon
  submission.
