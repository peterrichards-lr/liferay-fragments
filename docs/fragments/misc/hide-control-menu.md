# Hide Control Menu

The **Hide Control Menu** fragment is a specialized component within the Liferay
Fragments library.

## Overview

A utility to conditionally hide the Liferay control menu for specific users.

## Key Features

- **Meridian Ready**: Built using theme tokens for consistent spacing and color.
- **Accessible**: Follows Liferay accessibility standards.
- **Responsive**: Mobile-first design.
