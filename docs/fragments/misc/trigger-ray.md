# Trigger Ray

The **Trigger Ray** fragment is a specialized component within the Liferay
Fragments library.

## Overview

A specialized utility for triggering custom events within the Ray interaction
system.

## Key Features

- **Meridian Ready**: Built using theme tokens for consistent spacing and color.
- **Accessible**: Follows Liferay accessibility standards.
- **Responsive**: Mobile-first design.
