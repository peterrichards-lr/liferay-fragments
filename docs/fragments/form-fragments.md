# Form Fragments

An extensive collection of enhanced form components that extend the standard
Liferay DXP form capabilities with advanced UI patterns and data integration.

## Autocomplete (Object)

A text input that provides real-time suggestions based on data from a Liferay
Object. It is highly configurable, allowing for custom styling of the suggestion
list and background colors.

## Autocomplete (Picklist)

Similar to the Object version, but specifically designed to work with Liferay
Picklists, providing a polished autocomplete experience for predefined data
sets.

## Star Rating

A visual rating component that allows users to select a value by clicking on
stars. It supports both static ranges (e.g., 1-5 stars) and dynamic ranges based
on picklist options.

## Toggle Switch

A modern alternative to the standard checkbox, providing a clear "on/off" visual
state. It is fully accessible and integrates with the Meridian theme's primary
colors.

## Range Slider

Provides a visual slider for selecting a numeric value within a defined range.
It includes a real-time display of the currently selected value.

## Segmented Numeric

A specialized numeric input designed for entering values in a segmented format,
often used for reference numbers or formatted codes.

## Confirmation Field

Designed to ensure data integrity by requiring the user to enter the same value
twice (e.g., for email or password confirmation). It includes built-in
validation to check for mismatches.

## User Field

A specialized field for selecting or displaying user-related information, often
used in scenarios where a record needs to be associated with a specific Liferay
user.

## Relationship Fields

- **Hidden Relationship**: Allows for the silent association of records with a
  specific parent object or relationship ID.
- **URL-Populated Hidden Relationship**: Automatically pulls a relationship ID
  from a URL parameter, facilitating seamless record creation in nested or
  linked contexts.

## Layout & Action

- **Listbox Multiselect**: An enhanced multi-select component for handling large
  lists of options efficiently.
- **Submit Button**: A customizable submit button with support for different
  styles, sizes, and inverted colors, fully integrated with the Liferay form
  lifecycle.
