# Remote App Utilities

This collection provides tools and wrappers for integrating remote applications and external content into Liferay DXP.

## Liferay Iframer

The **Liferay Iframer** is an enhanced iframe component designed for embedding external web applications or pages.

### Key Features

- **Responsive Sizing**: Configurable height and width with responsive presets.
- **Dynamic Resizing**: Built-in support for listening to resize events from the child frame.
- **Security Control**: Easily configure `sandbox` and `allow` attributes for restricted environments.
- **Visual Polish**: Includes configurable borders, shadows, and loading states to match your Liferay theme.
