# Dashboard Container

The **Dashboard Container** fragment is a layout-focused component designed to host multiple widgets or charts in a responsive grid.

## Key Features

- **Responsive Grid**: Built-in support for different column layouts across desktop, tablet, and mobile.
- **Themed Borders**: Optional cards or shadows using Meridian CSS tokens.
- **Interactive States**: Supports loading indicators while child components are fetching data.

## Configuration

- **Layout Style**: Choose between single column, multi-column, or masonry.
- **Padding/Margin**: Configurable spacing using theme spacers (`var(--spacer-*)`).
