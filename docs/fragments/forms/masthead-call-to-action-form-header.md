# Masthead Call To Action Form Header

The **Masthead Call To Action Form Header** fragment is a specialized component
within the Liferay Fragments library.

## Overview

A visually rich masthead designed to sit above a lead generation form.

## Key Features

- **Meridian Ready**: Built using theme tokens for consistent spacing and color.
- **Accessible**: Follows Liferay accessibility standards.
- **Responsive**: Mobile-first design.
