# Generate Form Session ID

The **Generate Form Session ID** fragment is a utility component that creates
and initializes a unique session identifier.

## Key Features

- **UUID Generation**: Creates cryptographically strong unique identifiers.
- **Persistence**: Ensures the ID remains consistent throughout the user's
  browser session.
- **Silent Operation**: Operates in the background without affecting the page
  layout.

## Configuration

- **Storage Type**: Choose between session storage or local storage for the ID.
