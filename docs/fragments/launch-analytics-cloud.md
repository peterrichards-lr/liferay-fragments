# Launch Analytics Cloud

The **Launch Analytics Cloud** fragment is a utility component that facilitates the integration and manual triggering of Liferay Analytics Cloud events.

## Key Features

- **AC Integration**: Simplifies the process of sending custom tracking events.
- **Visual Button**: Optional themed button to trigger the event.
- **Silent Trigger**: Can also be configured to trigger automatically based on user interaction.

## Configuration

- **Event Name**: The name of the custom event to track.
- **Payload JSON**: Additional data to send with the event.
