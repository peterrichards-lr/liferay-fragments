# Profile Detail (Dashboard)

The **Profile Detail (Dashboard)** fragment is a specialized component within
the Liferay Fragments library.

## Overview

A compact version of the profile detail view designed for dashboard grids.

## Key Features

- **Meridian Ready**: Built using theme tokens for consistent spacing and color.
- **Accessible**: Follows Liferay accessibility standards.
- **Responsive**: Mobile-first design.
