# Profile Summary (Deprecated)

> **Warning**: This fragment is now **deprecated**. It relies on the legacy
> `serviceLocator` pattern which is restricted in many Liferay environments for
> security reasons. Users are encouraged to use modern **Data Providers** or
> **Headless APIs** to fetch user profile information.

The **Profile Summary** fragment provides a structured view of a user's basic
identity information.
