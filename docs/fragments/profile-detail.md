# Profile Detail

The **Profile Detail** fragment provides a comprehensive view of a user's account information, including custom fields and system attributes.

## Key Features

- **Dynamic Field Discovery**: Automatically lists available user attributes.
- **Custom Field Support**: Renders Liferay Expando (Custom Field) values.
- **Themed Layout**: Uses a clean, tabular, or list-based layout with Meridian tokens.

## Configuration

- **Excluded Fields**: A list of standard user fields to hide from the view.
- **Layout Mode**: Choose between single column or two-column grid.
