# Linear Gradient Container (Custom)

The **Linear Gradient Container (Custom)** fragment is a specialized component
within the Liferay Fragments library.

## Overview

An advanced gradient container with manual CSS override support.

## Key Features

- **Meridian Ready**: Built using theme tokens for consistent spacing and color.
- **Accessible**: Follows Liferay accessibility standards.
- **Responsive**: Mobile-first design.
