# Vertical Bar

A simple layout separator or vertical bar utility commonly used in header menus and layouts.

## Configuration

| Field                   | Description                                | Default |
| :---------------------- | :----------------------------------------- | :------ |
| **Enable Vertical Bar** | Determines if the vertical bar is visible. | `true`  |

## Usage

1. **Placement**: Add this fragment to a Header or Layout column.
2. **Settings**: Toggle the vertical bar visibility in the configuration panel.
