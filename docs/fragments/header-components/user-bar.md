# User Bar

A compact user identity and action bar for site headers, typically used to show
the current user's profile and quick links.

## Visuals

![User Bar](../../images/header-user-bar.png)

## Key Features

- **Meridian Ready**: Built using theme tokens for consistent spacing and color.
- **Accessible**: Follows Liferay accessibility standards.
- **Responsive**: Mobile-first design.
