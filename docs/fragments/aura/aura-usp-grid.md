# Aura - USP Grid

## Overview

The `aura-usp-grid` fragment (Unique Selling Proposition) highlights four distinct features or benefits in a 4-column horizontal grid. It is an excellent component for showcasing primary brand values or product highlights.

## Configuration

This fragment contains no configuration options in `configuration.json`.

## Usage & Behavior

The layout is built with a `grid grid-4` parent container holding four `aura-card` articles.
Editable regions for each of the four cards:

- `usp1_title` to `usp4_title` (text): The heading for the USP card.
- `usp1_text` to `usp4_text` (text): A short descriptive text line.

## Dependencies

- Heavily relies on `aura-container`, `grid`, `grid-4`, and `aura-card` CSS classes for the horizontal card layout.
