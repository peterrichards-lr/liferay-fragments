# Range Input

The **Range** fragment provides a stylized slider for selecting numeric values
within a defined interval.

## Key Features

- **Visual Feedback**: Displays the current value in a tooltip or adjacent label
  as the user slides.
- **Custom Min/Max**: Easily configurable boundaries and step increments.
- **Meridian Styling**: Modern slider design that matches the Meridian theme
  palette.

## Configuration

- **Minimum Value**: The lowest possible number.
- **Maximum Value**: The highest possible number.
- **Step**: The increment value between steps.
