# Hidden Relationship

The **Hidden Relationship** fragment is a utility for automatically linking form
submissions to a specific Liferay Object record without user intervention.

## Key Features

- **Silent Mapping**: Injects a specific record ID into a hidden field.
- **Workflow Integration**: Ideal for connecting feedback, comments, or
  applications to a parent record.
- **Dynamic Context**: Can be used alongside other fragments to pass contextual
  identifiers.

## Configuration

- **Relationship Name**: The internal name of the relationship in the Object
  definition.
- **Record ID**: The hardcoded or dynamic ID of the record to link.
