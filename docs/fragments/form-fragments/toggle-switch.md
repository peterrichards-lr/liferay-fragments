# Toggle Switch

The **Toggle Switch** fragment is a modern, accessible checkbox replacement for
binary settings.

## Key Features

- **Modern UI**: Polished sliding switch design.
- **Accessibility**: Full ARIA support and keyboard interaction (Space/Enter).
- **Meridian Theme**: Automatically inherits primary and secondary brand colors.

## Configuration

This fragment is designed to run inside form container environments, maps to standard input field fields, and inherits Liferay theme styling natively.
