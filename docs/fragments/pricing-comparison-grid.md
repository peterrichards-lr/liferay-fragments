# Pricing Comparison Grid

A visually rich comparison grid for service plans that pulls data from Liferay Objects or static config.

## Features
- **Interactive Toggles**: Switch between Monthly and Yearly pricing views.
- **Featured Card**: Automatically highlights the middle plan for high-conversion prominence.
- **Responsive Stacking**: Seamlessly transitions from side-by-side grid to vertical stack on mobile.
- **Meridian Styled**: Clean, modern design with deep shadows and refined typography.

## Configuration
- **Plans Data**: Semi-colon separated list of plan features and prices.
- **Highlight Color**: Custom accent color for the "Pro" or featured plan.
