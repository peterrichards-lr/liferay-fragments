# Content Map

The **Content Map** fragment provides an interactive OpenStreetMap integration for visualizing content locations or service points.

## Key Features

- **Map Integration**: Powered by OpenStreetMap and Leaflet.js.
- **Dynamic Markers**: Renders markers based on geo-coordinates from Liferay content or objects.
- **Custom Taxonomy Icons**: Map Liferay categories to specific Leaflet marker icons.
- **Service Access Policy**: Includes guidance on configuring necessary Liferay security policies.

## Configuration

- **Map Center**: Latitude and Longitude for initial view.
- **Zoom Level**: Default zoom magnification.
- **Category Icons**: Mapping of category IDs to SVG icon paths.
