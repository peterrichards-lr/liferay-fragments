# Grid Column

The **Grid Column** is the outermost structural fragment for Layout Components.
It ensures that cards nested within it are spaced and aligned correctly when
used inside a Liferay Grid row.

## Key Features

- **Standardized Spacing**: Applies consistent gutters and margins to match the
  Meridian theme.
- **Responsive Sizing**: Configurable column width for different screen sizes.

## Configuration

- **Column Size**: Set the bootstrap-style column width (e.g., 6 for half-width,
  4 for third-width).
