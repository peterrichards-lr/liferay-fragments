# My Rights

The **My Rights** fragment displays the current user's Liferay Roles and Permissions in a clean, informative list.

## Key Features

- **Security Visibility**: Helps users and administrators quickly verify assigned permissions.
- **Hierarchical View**: Distinguishes between Regular, Site, and Organization roles.
- **Themed Badges**: Uses Meridian color tokens to categorize different role types.

## Configuration

- **Group by Type**: Toggle whether to group roles by their scope.
- **Show Hidden Roles**: Option to display internal system roles.
