# Service Link Button

The **Service Link Button** is a specialized CTA component designed for consistent linking across service-oriented pages.

## Key Features

- **Themed Styling**: Utilizes Meridian CSS tokens for hover effects, transitions, and accessibility.
- **Icon Support**: Includes support for Lexicon/Clay icons.
- **Flexible Layout**: Easily integrates into card footers or hero sections.

## Configuration

- **Label**: The text displayed on the button.
- **URL**: The destination link.
- **Icon**: The name of the icon to display.
