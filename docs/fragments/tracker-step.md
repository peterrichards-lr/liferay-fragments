# Tracker Step

The **Tracker Step** fragment represents a single milestone within a parent "Tracker" container.

## Key Features

- **Status Indicators**: Visual cues for Pending, Active, Complete, and Error states.
- **Lexicon Icon Support**: Assign unique icons to each step to represent the milestone.
- **Interactive Labels**: Clickable step titles for navigating between process stages.

## Configuration

- **Status**: The current state of the step.
- **Icon**: The Clay icon to display.
- **Title**: The name of the milestone.
- **Description**: Optional text providing more detail about the step.
