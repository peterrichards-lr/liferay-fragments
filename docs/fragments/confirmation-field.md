# Confirmation Field

The **Confirmation Field** fragment ensures data integrity by requiring users to enter matching information across two inputs (e.g., "Confirm Email" or "New Password").

## Key Features

- **Real-time Validation**: Compares values as the user types and provides immediate visual feedback.
- **Configurable Error Messages**: Custom localized messages for mismatch states.
- **Meridian Styling**: Integrates seamlessly with standard Liferay form themes.

## Configuration

- **Target Field ID**: The ID of the field to compare against.
- **Error Message**: Localized text shown when values do not match.
