# Back Button

The **Back Button** fragment provides a simple, styled link or button that navigates the user to the previously visited page or a parent Liferay page.

## Key Features

- **Browser History Integration**: Can use `history.back()` for natural navigation.
- **Themed Styling**: Meridian-ready button design.
- **Icon Support**: Includes a back arrow or custom icon.

## Configuration

- **Target URL**: (Optional) A specific fallback page if no history is available.
- **Label**: The text shown on the button (e.g., "Back to List").
