# Toggle Switch

The **Toggle Switch** fragment is a modern, accessible checkbox replacement for binary settings.

## Key Features

- **Modern UI**: Polished sliding switch design.
- **Accessibility**: Full ARIA support and keyboard interaction (Space/Enter).
- **Meridian Theme**: Automatically inherits primary and secondary brand colors.

## Configuration

- **Label**: Description shown alongside the switch.
- **Checked Color**: Background color when the switch is ON.
