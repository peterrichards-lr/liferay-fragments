# Zone Layout

The **Zone Layout** fragment is a specialized component within the Liferay
Fragments library.

## Overview

A foundational layout component for defining structural zones on a page.

## Key Features

- **Meridian Ready**: Built using theme tokens for consistent spacing and color.
- **Accessible**: Follows Liferay accessibility standards.
- **Responsive**: Mobile-first design.
