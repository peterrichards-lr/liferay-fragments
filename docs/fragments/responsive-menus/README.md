# Responsive Menu Fragments

A collection of sophisticated, high-performance navigation components designed
to provide a seamless user experience across all device sizes.

## Core Navigation

- **Responsive Menu**: A powerful horizontal navigation fragment with built-in
  mobile responsiveness. It supports deep hierarchies, customizable dropdown
  behaviors, and integrates directly with Liferay's navigation menus.
- **Responsive Side Menu**: A specialized vertical navigation component, ideal
  for dashboards or secondary site sections. It features a collapsible
  "hamburger" menu for mobile views and supports multi-level nesting.

## Layout & Branding

- **Zone Layout**: A structural fragment that allows for the creation of
  distinct areas within a menu (e.g., Left, Center, Right). It uses Flexbox to
  ensure elements are aligned correctly and adapt to the available screen space.
- **Logo Zone**: A specialized container for brand assets. It provides easy
  placement for a site logo and can be configured to link back to the home page
  or a specific URL.

## Key Features

- **Freemarker (FTL) Implementation**: These fragments leverage the power of
  Freemarker to handle complex logic and deep menu recursion efficiently.
- **Adaptive UI**: Automatically switches between desktop (mega-menu style or
  standard dropdown) and mobile (collapsible accordion) views based on the
  device's screen size.
- **Advanced Configuration**: Includes extensive configuration options for
  transitions, colors, and layout behaviors, allowing for deep integration with
  a site's brand guidelines.
- **Meridian Theme Integration**: Fully supports Meridian CSS tokens for
  consistent typography, spacing, and accessibility.
