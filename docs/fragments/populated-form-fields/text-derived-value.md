# Text Derived Value

The **Text Derived Value** fragment is a specialized component within the
Liferay Fragments library.

## Overview

Calculates and populates a text field based on other form input values.

## Key Features

- **Meridian Ready**: Built using theme tokens for consistent spacing and color.
- **Accessible**: Follows Liferay accessibility standards.
- **Responsive**: Mobile-first design.
