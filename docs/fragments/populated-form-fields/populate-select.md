# Populate Select

The **Populate Select** fragment is a specialized component within the Liferay
Fragments library.

## Overview

Automatically populates a select input with options from a session or storage
value.

## Key Features

- **Meridian Ready**: Built using theme tokens for consistent spacing and color.
- **Accessible**: Follows Liferay accessibility standards.
- **Responsive**: Mobile-first design.
