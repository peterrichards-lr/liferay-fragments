# Customer Registration

The **Customer Registration** fragment provides a polished interface for new users to create accounts, with support for Liferay's self-service registration features.

## Key Features

- **Themed Design**: Styled to match the Meridian theme.
- **Link Mapping**: Configurable target for the "Register" CTA.
- **Visual Feedback**: Displays clear messages during the registration process.

## Configuration

- **Register Page URL**: The destination for the registration link.
- **Menu Text**: Customizable label for the registration entry point.
