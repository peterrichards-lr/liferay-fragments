# Date Display (Static)

The **Date Display (Static)** fragment is a simple utility for displaying a
hardcoded or manually entered date with consistent formatting across a page.

## Key Features

- **Manual Input**: Allows Page Editors to set a specific date value.
- **Themed Rendering**: Consistent styling with other date-related components.

## Configuration

- **Value**: The date to be displayed.
- **Format**: The desired date string format.
