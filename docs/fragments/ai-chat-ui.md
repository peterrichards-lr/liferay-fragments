# AI Assistant Chat UI

A polished messaging interface designed for integrating with AI backend extensions.

## Features
- Modern, responsive chat bubble interface.
- Support for "Assistant" and "User" message roles.
- Animated typing indicators.
- Seamless integration with Liferay Client Extensions.
- Accessibility-first design with `aria-live` regions.

## Configuration
- **Backend URL**: The endpoint of the AI Client Extension.
- **User Bubble Color**: Custom theme color for user messages.
- **Welcome Message**: Configurable initial greeting (via editable text).
