# Profile Summary

The **Profile Summary** fragment provides a high-level overview of the user's account, typically used in dashboard headers or sidebar widgets.

## Key Features

- **User Avatar**: Displays the user's Liferay profile picture.
- **Identity Details**: Shows the user's full name, job title, and organization.
- **Compact View**: Optimized for small containers or mobile layouts.

## Configuration

- **Show Email**: Toggle the visibility of the user's email address.
- **Avatar Size**: Choose from predefined size tokens.
