# Icon Button

The **Icon Button** is a minimalist, accessible button component that focuses on iconography for actions.

## Key Features

- **Clay Icons**: Supports the full set of Liferay Lexicon/Clay icons.
- **Accessibility**: Enforces the use of `aria-label` for screen reader users.
- **Hover Effects**: Polished interactive states using Meridian tokens.

## Configuration

- **Icon Name**: The name of the icon to display.
- **Aria Label**: Essential descriptive text for the action.
- **Color**: Choice of primary, secondary, or neutral button styles.
