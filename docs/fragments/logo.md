# Logo

The **Logo** fragment provides a highly configurable way to display and manage the site or organization brand in the header.

## Key Features

- **Dynamic Site Branding**: Can automatically pull the logo configured in Liferay Site Settings.
- **Manual Overrides**: Option to provide a custom image and alt text specifically for the header.
- **Responsive Sizing**: Configurable max-width and height to ensure brand consistency across devices.
- **Link Integration**: Automatically wraps the logo in a link to the site's homepage.

## Configuration

- **Site Name Toggle**: Choose whether to display the site name alongside the logo.
- **Alt Text**: Custom alternative text for accessibility.
- **Max Width/Height**: Constraints for the logo image.
