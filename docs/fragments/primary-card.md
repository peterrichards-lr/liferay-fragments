# Primary Card

The **Primary Card** fragment is a foundational layout component for displaying content in a structured, visually appealing container.

## Key Features

- **Meridian Styling**: Full integration with theme tokens for shadows, borders, and hover states.
- **Drop Zone Support**: Contains internal drop zones for nesting other fragments like buttons or icons.
- **Responsive Sizing**: Adapts smoothly to its parent container width.

## Configuration

- **Padding**: Adjust the internal spacing of the card.
- **Shadow Level**: Choose the intensity of the card's depth effect.
